📝 CSV Format Rules:

Columns required: district, year, examType, section, q, optA, optB, optC, optD, answer.
district: (e.g. pune), year: (e.g. 2024)
examType: "police_bharti" or "srpf"
section: "math", "gk", "reasoning", or "marathi"
answer: 0, 1, 2, or 3 (0=A, 1=B, 2=C, 3=D)

📝 JSON Format Example:

{
  "district": "pune",
  "year": 2024,
  "examType": "police_bharti",
  "sections": {
    "math": [
      { "q": "Question text?", "options": ["A","B","C","D"], "answer": 0 }
    ],
    "gk": [...],
    "reasoning": [...],
    "marathi": [...]
  }
}
// answer: 0=A, 1=B, 2=C, 3=D (zero-indexed)

एक छोटीशी मदत

ही वेबसाईट AI च्या मदतीने तयार करण्यात आली आहे आणि सध्या ती मोफत होस्टिंगवर चालवली जात आहे. जे पोलीस भरतीचे विद्यार्थी महागडी पुस्तके किंवा टेस्ट सिरीज घेऊ शकत नाहीत त्यांच्यासाठी हे एक मोफत व्यासपीठ आहे.

तुमच्या मदतीमुळे हे प्लॅटफॉर्म आणखी प्रगत करण्यासाठी व जास्तीत जास्त गरीब विद्यार्थ्यांपर्यंत पोहोचवण्यासाठी मदत होईल. धन्यवाद! 🚔
